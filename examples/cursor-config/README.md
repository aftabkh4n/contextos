# ContextOS with Cursor

Cursor support is on the roadmap. Cursor added MCP support in early 2025 and
the protocol is compatible, but we have not yet run the manual end-to-end test
that would let us write verified setup instructions here.

Once that test is done, this file will be updated with the exact configuration
steps. For now, if you want to try it, refer to the
[Cursor MCP documentation](https://docs.cursor.com/context/model-context-protocol)
for how to register an MCP server, and use the server command from
`examples/claude-code-config/README.md` as the command to run.

If you get it working, please open an issue or PR with what you found.
